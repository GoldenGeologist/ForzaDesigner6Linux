This is very basic because I like to make stuff for myself, so I dont usually think of making it easy for someone new using it. I will try to explain things and if its needed I can always make an installer in the future.

1. Create a folder and name it something like ForzaDesigner6
2. Drag the .zip file into the newly created folder
3. Extract the files into the folder
4. Open the "dist" folder
5. Right click anywhere inside the folder and select "Open In Terminal"
6. type `./app`

You did it! This works in most distros, but if there isn't an option to "Open In Terminal" you're probably way better with linux than I am. But just in case, `cd /your/dir/here/ForzaDesigner6/dist` will change the directory to the dist folder in terminal. Just change the first parts to where ever you put the ForzaDesigner6 folder.
